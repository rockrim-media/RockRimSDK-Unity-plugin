# RockRimSDK-Unity-plugin
Releases of the RockRimSDK Unity plugin

# Обзор плагина RockRimSDK 

RockRimSDK  — это Unity-плагин, предназначенный для интеграции серверной платформы RockRim в Unity проекты.  
Плагин предоставляет API для инициализации проекта, работы с магазином и управления покупками.

## Назначение

RockRim SDK упрощает интеграцию Unity проекта с backend-сервисами и предоставляет готовые инструменты для:

- определение страны платёжного аккаунта
- инициализация проекта;
- управление токенами авторизации;
- получение списка товаров магазина;
- оформление покупок;
- получение и потребление приобретённых товаров.

## Архитектура

Плагин построен по модульному принципу.  
Каждый модуль отвечает за отдельный функциональный блок и используется напрямую из C#-скриптов Unity.

## Совместимость

- Unity 2020 и выше
